Caret: 오픈소스 AI 개발 파트너

Caret은 AI 에이전트, 로컬 LLM, 실시간 협업 기능을 하나의 VS Code 확장 프로그램에 **통합하는 것을 목표로 합니다.** 투명하고 제어 가능한 AI 지원을 원하는 개발자를 위해 **만들어진 오픈소스 프로젝트입니다.**

왜 Caret인가요?

기존 AI 코딩 도구의 "블랙박스" 같은 동작 방식이나 제한적인 커스터마이징에 답답함을 느끼셨나요? Caret은 AI 에이전트의 작동 방식을 투명하게 이해하고, 개발자의 필요에 맞게 직접 제어하며, 원하는 LLM을 자유롭게 연결할 수 있는 환경을 제공하여 AI와 신뢰하는 동료처럼 협업하는 경험을 목표로 합니다.

핵심 가치 (목표)

  • 개발자 주도 오케스트레이션 (.caretrules 사용)
  • 멀티 에이전트 & RAG (Alpha 슈퍼바이저 + 전문 에이전트) (개발 목표)
  • Vibe Coding UX (CRDT 실시간 동기화) (개발 목표)
  • 모델 선택의 자유 – OpenAI, Groq, Gemini 또는 로컬 sLLM (지원 예정 포함)
    B2B sLLM 구축 및 컨설팅 지원 예정

핵심 모듈 (목표 구조)

Caret은 다음과 같은 모듈 구조를 목표로 개발되고 있습니다:

패키지역할
caret-core프롬프트 & 컨텍스트 엔진 (RAG, 캐시)
caret-agent4 + 1 모드 (Arch / Dev / Rule / Talk / Empty)
caret-ui-lite터미널 & 웹 뷰어, CRDT 동기화
caret-dev-kit로그, 프롬프트 실험실, 시각화

로드맵 주요 내용

  • 현재 – 초기 개발 및 리팩토링 단계 (v0.x)
  • 2025년 3분기 – 실시간 협업 MVP, Alpha 슈퍼바이저
  • 2025년 4분기 – Caret Pro 베타 (고급 RAG, 대시보드) + GCP 크레딧 매칭
  • 2026년 – 온프레미스 B2B sLLM 클러스터 & 컨설팅 패키지 출시

빠른 시작

git clone https://github.com/aicoding-caret/caret-zero.git
cd caret-zero
pnpm install && pnpm run build:vsix
# VS Code → 확장 프로그램 → VSIX에서 설치

프로젝트 루트에 .caretrules 파일을 생성하여 모드와 규칙을 정의하세요.
